Welcome to the Amiibo Collection!

This application is designed to help collectors manage their Amiibo collections.

What is an Amiibo?

An Amiibo is a collectible figurine of a Nintendo character from various series. Amiibos are very popular to collect, and this application helps collectors manage and track their collections.

This application allows you to:

  • Add new Amiibos based on name, series, condition, and ownership.
  • View a list of all the Amiibos in your collection.
  • Search for specific Amiibos using name, series, condition, and ownership.
  • Edit your collection list if things change in your collection.
